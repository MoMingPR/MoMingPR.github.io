创建博客历程
2024-05-29 17:38:18 #技术

首先要确认是否安装了Node.js,git.

直接上他们的官网选择符合的下载安装，全局配置。

第二步本地新建一个名为Blog的文件夹，在文件夹里右键启用 Git Bash Here

初始化hexo

hexo init

生成本地的hexo页面

hexo s

上传到Github托管

修改_config.yml文件

将deploy下的repository修改为你的github托管地址，注意空格

安装hexo-deployer-git自动部署发布工具

npm install hexo-deployer-git –save

生成静态页面文件

hexo g

运行本地服务查看

hexo s

在GitHub上创建个人github.io
仓库

上传静态页面文件

hexo d

‘hexo cl’命令用于清除缓存文件（db.json）和已生成的静态文件（public）。

例如：在更换主题后，如果发现站点更改不生效，可以运行该命令。

新建博客

hexo new post “新建博客文章名”
source文件夹下的_posts文件夹中编辑.md文件，推荐用软件Typora编辑，因为好排版。
